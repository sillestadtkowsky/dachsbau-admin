== Description ==
Dieses PLugin stellt ein Admin-Menü im Dashboard von WordPress zur Verfügung. Das Menü enthält eine Hauptseite ("Dachsbau-Admin") und vier Untermenüs: "Mitgliederliste bearbeiten", "Mitgliederliste importieren", "Gelöschte Buchungen" und "Konfiguration".
Das Ziel dieses Plugin ist es, dem Administrator des WordPress-basierten Dachsbau-Systems eine einfache Möglichkeit zu bieten, Mitgliederlisten zu importieren, zu bearbeiten sowie Buchungen und Konfigurationen zu verwalten und 
eine automatische Buchungssicherung durchzuführen zu lassen.


== Installation ==
Upload the plugin files to the /wp-content/plugins/dachsbau-karow directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress.
